# Homework

实现一个命令行文本计数统计程序。能正确统计导入的纯英文txt文本中的字符数，单词数，句子数。
将需要测试的文本输入test.txt文本中，运行wc.cpp文件，就可以得到输出结果。
文件列表:test.txt               测试文本
        words.txt              单词数目结果输出文本
        sentences.txt          句子数目结果输出文本
        characters.txt         字符数目结果输出文本
        lines.txt              行数数目结果输出文本
        wc.cpp                 运行程序
